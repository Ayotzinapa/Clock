# Clock

A lightweight Windows console clock written entirely in x86-64 Assembly using NASM and the Win32 API.
### IF something isn't working feel free to edit this project.

## Build

### Requirements

- Windows x64
- NASM
- MSYS2 UCRT64 / MinGW-w64
- mingw32-make

### Compile

```bash
mingw32-make
